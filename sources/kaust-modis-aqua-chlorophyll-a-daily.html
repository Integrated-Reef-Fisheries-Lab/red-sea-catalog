AQUA/MODIS Chlorophyll-a Daily L3m Ocean Color — Red Sea

Daily NASA AQUA/MODIS Level-3 mapped (L3m) ocean color chlorophyll-a concentration granules covering the Red Sea, downloaded per-scene from NASA OceanColor/OB.DAAC and archived internally in three parallel forms: an extracted directory of daily NetCDF granules, a tar bundle of the same granules, and a single time-merged ('combined') NetCDF file spanning the full record. Standard OC3/OCx chlorophyll-a algorithm product (variable 'chlor_a').

Known issues: n_rows for the combined NetCDF is inferred from the granule count of the daily-file resources, not verified by opening the file (no NetCDF-reading tool was available in this environment).; File count (7123) does not equal a full daily count over 2005-2026 due to cloud-masked/no-data days already excluded upstream at NASA.; Files after ~2026-01 carry a .NRT (near-real-time) suffix and may later be replaced by reprocessed standard files; duplicates/overlap with final files not checked.

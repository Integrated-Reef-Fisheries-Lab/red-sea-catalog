ReefNet: Global Coral Reef Image Annotation Dataset (v1.0)

ReefNet is a large-scale, taxonomically enriched collection of annotated coral reef images compiled from many CoralNet sources worldwide, built for training and benchmarking machine-learning coral classification models. It includes a consolidated global label set, per-source metadata, and a dedicated Red Sea test partition (the 'RSG-TestDataset') contributed by Red Sea Global, collected in the Al-Wajh lagoon (~26.4N, 36.4E) in 2022 and used as a held-out evaluation set in the ReefNet paper. The dataset as a whole is global in scope; only the RSG-TestDataset partition is Red-Sea-specific.

CoralNet Crawler (ReefNet image/annotation scraping toolkit)

A Python utility toolkit ('CoralNet Crawler') developed as part of the ReefNet project to scrape source metadata, point annotations, and images from CoralNet (coralnet.ucsd.edu), a community coral-reef photo-annotation platform. This is code/tooling used to construct the ReefNet dataset, not a dataset itself -- no scraped image or annotation output is present in this folder (only the scripts and an input sources list). Catalogued here for provenance completeness, linking to the reefnet-coral-image-dataset record it helped produce.

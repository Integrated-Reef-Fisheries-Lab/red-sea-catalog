Global Fishing Watch — AIS Apparent Fishing Effort, Saudi Arabia EEZ (2012–2025)

Gridded, low-spatial-resolution daily apparent fishing hours derived from AIS vessel tracking, aggregated by flag state and (intended) gear type within the Saudi Arabia EEZ region, downloaded via the gfwr R client per the accompanying processing script (GFW.R). Covers both the Red Sea and Arabian Gulf coasts of Saudi Arabia.

Known issues: The Geartype column is empty ('NA') for every row in this extract, even though the processing script (GFW.R) requested group_by = 'FLAGANDGEARTYPE'; gear-type breakdowns are not usable from this file as-is.; 'Vessel IDs' is a count of unique contributing vessels, not an actual vessel identifier — explicitly noted as a gotcha in GFW.R's inline comments.; Coverage is the full Saudi Arabia EEZ (GFW EEZ region code 'SAU'), which spans both the Red Sea and Arabian Gulf coasts (observed longitude range 34.5-51.6°E, latitude 16.4-29.4°N); records east of roughly 42-43°E fall in the Arabian Gulf, not the Red Sea, and should be filtered out (e.g. by bbox or by geometry) for Red-Sea-specific analyses.; Low spatial resolution ('LOW' setting in the gfwr query) — coordinates are coarse grid-cell centroids, not vessel tracks.; This is a static, point-in-time extract (per GFW.R, pulled through 2026-03-27 request date, with data present through Nov 2025); it will not reflect more recent activity unless the script is rerun.; License terms for redistribution were not verified locally; consult Global Fishing Watch's data-download terms before further use/sharing.
